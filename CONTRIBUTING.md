# Contributing

Thanks for considering a contribution.

## Development

```bash
npm install
npm run dev
```

Before opening a pull request, run:

```bash
npm run lint
npm run typecheck
npm run build
npm run test
```

## Guidelines

- Keep changes focused and avoid unrelated refactors.
- Follow the existing TypeScript, Vue, and SCSS style.
- Do not commit `.env`, secrets, personal data, private screenshots, or generated build output.
- Use safe placeholder values in docs and examples.
- Add screenshots or recordings for visual UI changes.

## Pull Requests

Please include:

- A short summary of the change
- Verification commands you ran
- Any environment variable or deployment changes
- Screenshots for visual changes
